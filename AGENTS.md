# Agent 行为准则 — Public Opinion Forecasting

## 核心原则

- 任何任务开始前，先与用户充分沟通，明确需求细节，严禁擅自揣测。
- 若需求存在模糊、歧义或缺失信息，必须暂停执行，主动提问澄清。

---

## 前端设计系统规范（全局）

### 1. 风格定位

本项目前端采用统一的**"暖阳人文"浅色设计系统**：

- 整体风格温暖、明亮、有人文温度，与油画封面图形成统一视觉语言。
- 拒绝纯冷峻科技风和深黑色调，所有页面（封面页 + 内页）均使用浅色暖调背景。
- 数据可视化区域通过卡片容器、边框层次和强调色来建立信息层级，而非依赖深色背景。

### 2. 色彩系统

| Token | 色值 | 用途 |
|-------|------|------|
| `--canvas` | `#FAF7F2` | 页面全局背景（极浅奶油白） |
| `--surface-1` | `#FFFFFF` | 卡片、面板、侧边栏背景 |
| `--surface-2` | `#F5F0E8` | 悬停状态、次要面板、选中标签 |
| `--surface-3` | `#F0EBE3` | 三级表面、下拉菜单背景 |
| `--surface-4` | `#E8E3DA` | 最深 lifted surface（极少用） |
| `--hairline` | `#E2DDD4` | 卡片边框、分割线 |
| `--hairline-strong` | `#D5CFC6` | 输入框聚焦、强边框 |
| `--hairline-tertiary` | `#C8C2B8` | 嵌套表面边框 |
| `--primary` | `#2563EB` | 主按钮、强调色、进度条、链接（钴蓝，取自油画天空） |
| `--primary-hover` | `#1D4ED8` | 按钮悬停、链接悬停 |
| `--primary-focus` | `#1E40AF` | 聚焦环 |
| `--ink` | `#2D2926` | 主标题、正文（深暖炭） |
| `--ink-muted` | `#6B6560` | 副标题、次要文字（暖灰） |
| `--ink-subtle` | `#9E9890` | 辅助文字、占位符（浅暖灰） |
| `--ink-tertiary` | `#B5AFA6` | 禁用状态、脚注（更浅暖灰） |
| `--success` | `#059669` | 成功状态、positive sentiment（嫩绿，取自油画草地） |
| `--warning` | `#D97706` | 警告状态（暖橙） |
| `--error` | `#DC2626` | 错误状态（珊瑚红） |

**色彩提取来源**：所有暖色调均从封面油画中提取——天空→钴蓝、草地→嫩绿、阳光→暖橙、树干→深暖炭、云朵→奶油白。

### 3. 封面页（Landing Page）规范

- **布局**：左右分栏（桌面端：左文右图），上下分栏（移动端：上文下图）。
- **背景**：`var(--canvas)` `#FAF7F2`。
- **标题**：`clamp(36px, 5vw, 56px)`，字重 700，`var(--ink)`。
- **副标题**：18px，`var(--ink-muted)`，行高 1.6。
- **主按钮**：`var(--primary)` 背景，白色文字，阴影 `0 4px 14px rgba(37, 99, 235, 0.25)`，悬停 `var(--primary-hover)`。
- **封面图**：圆角 16px，阴影 `0 20px 60px rgba(45, 41, 38, 0.12)`，**禁止**加 `filter: invert()` 或降低透明度。
- **容器**：最大宽度 1280px，居中，左右内边距 48px（移动端 24px）。

### 4. 内页（Dashboard / Graph / Simulation / Report 等）规范

- **整体布局**：左侧固定侧边栏（240px）+ 右侧主内容区。
- **侧边栏**：背景 `var(--surface-1)` `#FFFFFF`，右边框 `var(--hairline)` `#E2DDD4`。
- **顶部栏**：高度 56px，背景 `var(--canvas)` `#FAF7F2`，底边框 `var(--hairline)` `#E2DDD4`。
- **卡片**：背景 `var(--surface-1)` `#FFFFFF`，边框 `var(--hairline)` `#E2DDD4`，圆角 12px，内边距 24px。
- **按钮**：
  - Primary：`var(--primary)` 背景，白色文字，圆角 8px。
  - Secondary：`var(--surface-1)` 背景，`var(--ink)` 文字，`var(--hairline)` 边框。
- **表单输入**：背景 `var(--surface-1)`，边框 `var(--hairline)`，聚焦边框 `var(--primary-focus)`。
- **表格**：表头文字 `var(--ink-subtle)`，12px，大写，字间距 0.4px；行悬停 `rgba(45,41,38,0.04)`。
- **Badge**：
  - success: `rgba(5,150,105,0.12)` 背景，`#047857` 文字
  - warning: `rgba(217,119,6,0.12)` 背景，`#B45309` 文字
  - error: `rgba(220,38,38,0.12)` 背景，`#B91C1C` 文字
  - info: `rgba(37,99,235,0.12)` 背景，`#1D4ED8` 文字

### 5. 排版

- **字体**：`Inter, -apple-system, BlinkMacSystemFont, sans-serif`。
- **行高**：正文 1.5，标题 1.1–1.2。
- **字间距**：标题 `-0.02em` 至 `-0.03em`，正文默认。
- **最小字号**：正文 14px，辅助文字 12px，按钮 14px。

### 6. 响应式断点

| 断点 | 宽度 | 关键变化 |
|------|------|----------|
| Desktop-XL | 1440px | 默认桌面布局 |
| Desktop | 1280px | 3 列卡片网格 |
| Tablet | 1024px | 3 列 → 2 列 |
| Mobile-Lg | 768px | 侧边栏收拢为图标栏；封面页变为上下布局 |
| Mobile | 480px | 单列；封面标题缩小 |

### 7. 图标与素材

- **图标**：统一使用 SVG（Lucide / Heroicons 风格），禁止使用 Emoji 作为图标。
- **封面图**：仅使用项目根目录下经确认的油画风格封面（`frontend/cover.jpg`），明亮欢快色调，印象派笔触，3:2 横向比例。
- **产品截图**：内页如需展示产品界面，使用浅色圆角面板（16px 圆角，`#FFFFFF` 背景）包裹。

### 8. 动效规范

- **微交互时长**：150–300ms。
- **缓动**：`ease-out` 用于进入，`ease-in` 用于退出。
- **可打断**：所有动画必须可被打断，用户操作优先。
- **减少动效**：尊重 `prefers-reduced-motion`。
